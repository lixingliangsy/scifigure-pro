What Are AI-Generated Scientific Figures?

AI-generated scientific figures are visual representations of data, concepts, or results created using artificial intelligence models. These tools use:

  • Diffusion models (like Stable Diffusion, DALL-E 3) for concept diagrams and illustrations
  • Large Language Models (like GPT-4, Claude) for code generation (Python/R) that creates charts
  • Specialized scientific AI (like SciFigure Pro) trained on millions of peer-reviewed figures

Common Mistakes to Avoid

Mistake 1: Trusting AI Output Without Verification

AI tools can hallucinate data or mislabel axes. Always double-check:

  • Axis labels and units
  • Data values (compare to your original data)
  • Statistical annotations (p-values, error bars)

Mistake 2: Using Low-Resolution Outputs

Journals require high resolution (minimum 300 DPI for print). Always export at the highest resolution available, then downscale if needed.

Mistake 3: Ignoring Journal Guidelines

Each journal has specific requirements for:

  • Figure dimensions (single column, 1.5 column, full page)
  • File format (TIFF, EPS, PDF)
  • Color mode (RGB for online, CMYK for print)
  • Font size (minimum 6pt)
